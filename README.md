# Coursera_GettingAndCleaningData
# This readme.md describes how to use run_analysis.R
#1.	UnZip the data from the source below
#2.	https://d396qusza40orc.cloudfront.net/getdata%2Fprojectfiles%2FUCI%20HAR%20Dataset.zip 
#3.	Make sure that the run_analysis.R and the data folder are in same working directory
#4.	open RStudio 
#5.      set working directory to where the relevant data and run_analysis.R file are copied 
#6.	Type source(“run.analysis.R) command in RStudio and hit enter
#7.	The script will create tidy_data.txt, tidy_data.csv, tidyData.WithMean.txt, tidyData.WithMean.csv files
#8.	To view the average of each variable for each subject and each activity - 
#       Type command View(mean.tidyData) and hit enter.
